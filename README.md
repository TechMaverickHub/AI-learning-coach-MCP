# AI-learning-coach-MCP
AI Learning Coach MCP + Streamlit Dashboard (Supabase + Groq + pgvector)  A full-stack Learning Coach system built using:  Supabase (pgvector) for content storage, semantic search &amp; embeddings  Groq LLM (llama3.1-8b-instant) for fast, low-latency digest generation  SentenceTransformer (all-MiniLM-L6-v2) for embedding generation
